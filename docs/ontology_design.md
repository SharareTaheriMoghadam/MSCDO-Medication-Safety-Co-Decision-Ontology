# Ontology Design

MSCDO follows a modular ontology engineering approach.

Core modules include:

- Medication Knowledge
- Patient Context
- Medication Risks
- Human–AI Collaboration
- Explainable AI
- Clinical Recommendations
- Prevention Strategies

The ontology aligns with:

- WHO ICPS
- SNOMED CT
- RxNorm
- MedDRA
- LOINC
- ATC

Reasoning is supported using OWL axioms and SWRL rules.
