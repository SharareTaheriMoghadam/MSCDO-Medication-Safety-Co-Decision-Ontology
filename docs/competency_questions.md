# Competency Questions

MSCDO was designed to answer questions such as:

1. Which medications have known adverse drug events?

2. Which patients require enhanced monitoring?

3. Which medication combinations are considered high risk?

4. Which patients require dose adjustment?

5. Which medication errors can be prevented?

6. Which patient characteristics increase medication-related risk?

7. Which recommendations require clinician review?

8. Which medications are associated with CYP450 interactions?
